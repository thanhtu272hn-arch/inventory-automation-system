\# Businees Automation Dashboard



A Businees Automation Dashboard built with Streamlit.



\## Features

\- Total Income

\- Total Expense

\- Net Profit

\- Profit Margin

\- Category Filter

\- Date Filter

\- Expense Breakdown

\- Monthly Profit Trend



\## Tech Stack

\- Python

\- Pandas

\- Streamlit



\## Run

streamlit run app.py
